Cascading Style Sheet종속형 시트 또는 캐스케이딩 스타일 시트(이하 CSS)는 마크업 언어가 실제 표시되는 방법을 기술하는 Style sheet language스타일 언어로,
HTML과 XHTML에 주로 쓰이며, XML에서도 사용할 수 있다.

W3C의 표준이고, 레이아웃과 스타일을 정의할 때의 자유도가 높다. 기본 파일명[2]은 style.css이다.

HTML가 웹사이트의 몸체를 담당한다면 CSS는 옷과 액세서리처럼 꾸미는 역할을 담당한다고 할 수 있다.
즉, HTML 구조는 그대로 두고 CSS 파일만 변경해도 전혀 다른 웹사이트처럼 꾸밀 수 있다.
